# ⭐️ baekjoon-js  
백준 문제 풀이 저장소 (JavaScript)

이 저장소는 제가 **하루에 조금씩이라도 꾸준히 알고리즘을 공부하고**,  
문제 해결 사고력을 키우기 위해 만든 백준 문제 풀이 기록 공간입니다.

JavaScript(Node.js)를 사용해 문제를 풀이하며,  
각 문제는 **입력 ➡ 로직 설계 ➡ 예외 처리 ➡ 출력** 순으로 작성합니다.

---

## 레포 운영 목적
- 꾸준한 문제 풀이 습관 만들기
- 문제 해결 능력 향상
- JavaScript 알고리즘 구현 연습
- 기술면접 대비 및 학습 기록 관리
---

## 앞으로의 계획
- 하루 1문제 이상 풀이
- 단계별 ➡ 알고리즘 분류별 확장
- 자주 틀리는 유형 정리
- 중급 단계에서는 TypeScript로 시도
